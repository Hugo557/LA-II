# Lenguajes y Autómatas II

Este repositorio contiene los proyectos y prácticas desarrolladas durante la materia **Lenguajes y Autómatas II**, incluyendo:

## Contenido

- **Analizador Léxico**  
  Implementación del analizador léxico que identifica tokens, palabras reservadas y símbolos válidos dentro de un lenguaje definido.

- **Analizador Sintáctico**  
  Desarrollo del analizador sintáctico que valida la estructura gramatical de las expresiones según una gramática formal.

- **Proyecto Final**  
  Integración completa del analizador léxico y sintáctico en un proyecto final que aplica los conceptos teóricos y prácticos de la asignatura.
